# dice-roguelite
## 在 Cocos Creator 中挂载并运行
1. 打开项目后，在层级管理器中新建一个空节点（例如命名为 `GameRoot`）。
2. 将 `assets/scripts/GameRoot.ts` 挂载到该节点，并在属性检查器中绑定 `rollButton`、`infoLabel`、`diceLeftLabel`、`baseHpLabel`、`enemyRoot`。
3. 在 Canvas 下创建一个 Button（文本写“掷骰”）和三个 Label，用于展示日志、剩余骰子次数与 Base HP。
4. 在 Canvas 下创建一个空节点命名为 `EnemyRoot`，用于生成敌人，并拖拽到 `GameRoot` 组件的 `enemyRoot` 属性。
5. （可选）创建一个空节点作为 `boardRoot`，以及一个空节点作为 `playerNode` 并绑定；未绑定时脚本会自动生成简单棋盘与棋子。
6. 点击运行后，按下“掷骰”按钮即可在日志里看到移动与事件效果。

## 当前棋盘与玩法
- 棋盘为 6×6 外圈路径，共 20 格，中心留空。
- 格子编号按顺时针排列：上边从左到右、右边从上到下、下边从右到左、左边从下到上。
- 玩家移动为循环路径，掷骰后位置 = (当前位置 + 点数) % 20。
- 探索阶段一共 3 次骰子，用完会自动进入战斗，波次全部清完后回到探索并补充 3 个骰子。
- 武器格默认在索引 3、8、13、18（从 0 开始），落到武器格可选择武器并放置到该格。
- 战斗阶段敌人从中心出生移动到外圈格子，抵达后会持续攻击 Base HP，直到被武器击败。
